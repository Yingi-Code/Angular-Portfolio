How does it works?

This component demonstrates how to implement reactive forms in angular. You can manage developers' team by adding new developer, viewing details, updating details, allocating or removing developer from project team, and deleting developer from the list.

The scope of work
  • Responsive UI Design concept.
  • Reactive forms, forms validation, and route guard.
  • Data structures - Arrays and Objects.
  • Angular services - Dependency injection.
