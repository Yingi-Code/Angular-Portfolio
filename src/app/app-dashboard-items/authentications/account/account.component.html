Basic Authentication (no token) JWT Authentication Auth0 Authentication
This section introduces various mechanisms in which authentication can be implemented using angular:
  • [ Basic Authentication ] - only username and password, no Token required.
  • [ JWT Authentication ] - requries username, password, and retrieval of Token.
  • [ Auth0 Authentication ] - to be viewed.
Description

This component lists vaious mechanisms and approaches in which Authentication can be implemented in angluar application. The mechanisms include standard authentication (username + password), JWT authentication, and Auth0 authentication. The authentication mechanisms are implemented by integrating the application with API.

  • Responsive UI/UX Design concept using (HTML5, CSS3, Bootstrap 5, and FontAwesome).
