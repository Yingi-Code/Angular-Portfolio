Description

This component demonstrates how to implement Cascading drop-downs in angular application to drill-down data to specific level. It's a three-level dependent drop-doown menu that allows the user to make couple of selections before retriieving the intended information.

The scope of work
  • Responsive UI/UX Design concept using (HTML5, CSS3, Bootstrap 5, and FontAwesome).
  • Template-driven forms, forms validation, and route guard.
  • Data structures - JSON Data.
