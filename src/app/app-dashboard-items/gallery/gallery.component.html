Description

This component demonstrates how to implement Photo gallery in angular application. The applications presents a list of all items in online photo-album in a thumbnail size. It allows the user to further view each photo in an original size by clicking on each photo. It also allows the user to zoom the image, navigate or slide images, and also download images.

The scope of work
  • Responsive UI/UX Design concept using (HTML5, CSS3, Bootstrap 5, and FontAwesome).
  • Data structures - Arrays and Objects.
  • Unit Test - Karma and Jasmine.
