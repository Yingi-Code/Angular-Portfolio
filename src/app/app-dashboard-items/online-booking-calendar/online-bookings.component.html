Description

This component demonstrates how to implement and customize Angular calendar in angular application. The calendar can be configured to display specific features depanding on the needs as required by the application. The customization includes the default content display format. eg, display items in a list-calendar-view or display items in a week-calendar-view. Users can view the details of an item by clicking or selecting that specific item in the calendar.

The scope of work
  • Responsive UI/UX Design concept using (HTML5, CSS3, Bootstrap 5, and FontAwesome).
  • Data structures - Arrays and Objects.
  • Unit Test - Karma and Jasmine.
