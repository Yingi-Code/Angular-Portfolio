Description

This component serves as a Dashboard or Control Panel for this portfolio. It presents easy-access to various features and functionalities of the portfolio. The status for each development progress very depending on whether the feature is just a concept , the development of a feature is in-progress , the feature is completed , which are represented in different colors.

The scope of work
  • The implementation of Responsive UI Design principles.
  • The development progress status for each listed feature.
